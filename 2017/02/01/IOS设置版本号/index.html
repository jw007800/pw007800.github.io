IOS设置版本号

原因
友盟上报版本号时只会取CFBundleVersion，即使已经设置了版本号CFShortVersionString，友盟也会取该版本号，这样会于之前定义的版本号发生冲突。
版本控制工具
通过agvtool来设置iOS端App版本号，参考：
https://developer.apple.com/library/ios/qa/qa1827/_index.html

在BuildSetting中选择Versioning，设置Versioning System为Apple Generic
在命令行中，进入项目目录，执行
agvtool next-version
即可叠加当前版本号。

设置新的版本号，执行
agvtool new-version #NEW_VERSION#
即可为当前项目设置新的版本号

自定义DailyBuild版本
在Info.plist中添加自定义项：DailyBuildVersion，数据类型为String，数据为${CURRENT_PROJECT_VERSION}
设置BundleVersion中得数据与Bundle versions string, short的数据保持一致。
