GO环境安装

发表于 2017-01-17 | 分类于 GO开发

安装

brew install GO

安装完成后可以命令 go version 查看版本

go env 查看环境

需要说明的是：

GOPATH:日常开发的根目录

GOBIN:是GOPATH下的bin目录

设置环境变量

vim .bash_profile

export GOROOT=/usr/local/Cellar/go
export GOPATH=/Users/songjie/Developer/GO
export GOBIN=$GOPATH/bin
export PATH=$PATH:$GOBIN

退出vim,使用source ~/.bash_profile即可完成对全局变量的配置了，可以在终端中输入go env查看配置后的效果:

需要注意的是安装之后运行会告诉你找不到env中找不到PATH

解决办法：preferences -> LiteEnv -> system.env

#GOROOT=/usr/local/Cellar/go

#GOARCH=amd64

/*PATH=$PATH 还是会提示找不到PATH*/

LITEIDE_GDB=/usr/local/bin/gdb
LITEIDE_MAKE=make
LITEIDE_TERM=/usr/bin/open
LITEIDE_TERMARGS=-a Terminal
LITEIDE_EXEC=/usr/X11R6/bin/xterm
LITEIDE_EXECOPT=-e
